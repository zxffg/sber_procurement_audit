## Этап 1. Сбор данных

Пперед сбором данных, мною был составлен полный список юр.лиц, которые относятся к группе компании Сбер. Вы можете ознакомиться с ним в файле
`exploratory_data.ipynb`.

### Обоснование выбора вертикали

Для анализа выбран **ИТ/финтех-контур экосистемы Сбера** — дочерние общества, работающие в сфере технологий, цифровых финансов и сервисов.
Выбор обусловлен тремя факторами:

1. **Активность закупок.** ИТ-компании закупают ПО, оборудование, облачные мощности регулярно и в значительных объёмах, что даёт статистически
достаточную выборку.
2. **Аналитическая ценность.** Закупки ИТ-сектора тесно коррелируют с внешними факторами (курс USD/RUB, ключевая ставка), что позволяет
строить содержательные гипотезы в Этапе 3.
3. **Доступность данных.** В отличие от головного ПАО Сбербанк (данные закрыты с 2022 г. по регуляторным основаниям), дочерние ИТ-структуры
публикуют закупки в коммерческой секции ЭТП.

Для составления полного перечня юридических лиц группы использован **Центр раскрытия корпоративной информации Интерфакс** ([e-disclosure.ru
(https://e-disclosure.ru)), предоставляющий данные в открытом доступе, поскольку отчётность по МСФО на официальном сайте Сбера содержит
ограниченные сведения о составе группы в условиях санкционного давления.

### Площадки из технического задания — результаты анализа

**[Сбербанк-АСТ](sberbank-ast.ru) с поддоменом `utp-` Универсальная торговая площадка - основной источник.**
[ЕИС](zakupki.gov.ru), [РТС-тендер](rts-tender.ru), [Росэлторг](roseltorg.ru), [ТекТорг](tektorg.ru), [ЗаказРФ](etp.zakazrf.ru), [ЛотОнлайн
(lot-online.ru), [ЕТП ГПБ](etpgpb.ru).

Большинство площадок из перечня задания не содержат актуальных данных по группе Сбер за 2024–2025 гг. по двум независимым причинам.

**Постановление Правительства РФ №301 от 06.03.2022**
С марта 2022 года организации, находящиеся под санкциями, **освобождены от обязательного размещения информации о закупках в ЕИС**. Поскольку
ПАО Сбербанк находится под западными санкциями, сведения о его закупках в публичном реестре ЕИС за 2024–2025 гг. **отсутствуют полностью.**
Данное ограничение существенно снижает прозрачность закупочной деятельности крупнейшего банка страны с государственным участием.

**Ограничение доступа к данным ЕИС с 01.01.2025**
С 1 января 2025 года **массовая выгрузка данных из ЕИС** стала возможна исключительно через **авторизованный API** вместо прежнего открытого
FTP-сервера. Ранее свободный доступ позволял аналитическим агрегаторам и исследователям получать данные без регистрации. Теперь для получения
массива данных необходима регистрация в ЕИС как юридическое лицо или ИП.

Агрегаторы (tenderer.ru и др.) прекратили полноценное обновление данных с начала 2025 г. Данные за этот период доступны только через прямой
запрос к API ЕИС (нужна регистрация). Коммерческие сервисы по типу [Контур.Закупки](https://zakupki.kontur.ru) не несут практической ценности.

Информация о победителях закупок недоступна в открытых источниках для 100% процедур выборки. Коммерческая секция ЭТП Сбербанк-АСТ не
предусматривает публикацию итогов для внешних пользователей. Закупки проводятся в ограниченном режиме. Анализ конкурентности и аномалий по
победителям возможен только при наличии доступа к внутренним системам группы Сбер.

### Воспроизведение процесса

Более подробно узнать о обезличивании данных и обогащении таблицы можно в файле `exploratory_data.ipynb`

**Шаг 1. Формирование списка ИНН**
1. Перейти на e-disclosure.ru
2. Найти ПАО Сбербанк → «Аффилированные лица» / «Дочерние общества»
3. Выгрузить список → взять ИНН каждого юрлица
4. Верифицировать ИНН через egrul.nalog.ru

**Шаг 2. Сбор данных с ЭТП Сбербанк-АСТ**
1. Перейти: utp.sberbank-ast.ru → Торги → Поиск
2. Фильтры:
   - Тип торгов: Коммерческие / SberB2B
   - ИНН заказчика: [ИНН целевой компании]
   - Период размещения: 01.01.2024 – 31.12.2025
3. Инструмент парсинга пагинации: Instant Data Scraper и ИИ-скрапер Web Scraper
